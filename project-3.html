Il NewYork Times

Rielaborazione della homepage del New York Times, utilizzando le API ufficiali per l'acquisizione dei dati.

Live Link

Project Overview

Questo progetto ha richiesto l'uso di tecnologie come React e Axios. L'esperienza pratica con questi strumenti mi ha permesso di acquisire competenze tecniche preziose e una comprensione approfondita del loro funzionamento.

L'integrazione di diverse API è stata una caratteristica saliente del mio progetto. Questo ha migliorato la tua capacità di manipolare e integrare diversi set di dati, competenza trasferibile a una vasta gamma di contesti.

Il lavoro su interfacce utente utilizzando React e CSS ha ampliato la mia comprensione del design dell'interfaccia utente. Hai appreso come replicare il design esistente e come migliorare l'esperienza dell'utente.

Infine, risolvere i problemi tecnici durante lo sviluppo del progetto mi ha permesso di acquisire competenze di problem solving. Ogni sfida superata mi ha reso più sicuro delle mie abilità e più preparato per progetti futuri.
