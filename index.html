Login phase with OAuth 2.0

Realizzazione di un processo di login utilizzando OAuth 2.0, con l'implementazione dell'estensione PKCE (Proof Key for Code Exchange). Questo lavoro ha consentito un'immersione profonda nella comprensione dei meccanismi di sicurezza implementati da PKCE.

Framework utilizzati: React, Keycloak e Spring.
